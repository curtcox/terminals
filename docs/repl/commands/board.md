# board

See `help board` for the typed command surface.

## Commands

- `board ls [board] [--json]`
- `board post <board> <text> [--json]`
- `board pin <board> <text> [--json]`

## Notes

- `board post` creates a standard board entry.
- `board pin` creates a pinned bulletin entry.

## Examples

```text
board ls family
board post family Need milk
board pin family Dinner in 10
```
