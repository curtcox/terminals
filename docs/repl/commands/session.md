# session

Typed generalized interactive session commands:

- `session ls`
- `session create <kind> <target>`
- `session show <session>`
- `session members <session>`
- `session join <session> <participant>`
- `session leave <session> <participant>`
- `session attach <session> <device-ref>`
- `session detach <session> <device-ref>`
- `session control request <session> <participant> [control-type]`
- `session control grant <session> <participant> [granted-by] [control-type]`
- `session control revoke <session> <participant> [revoked-by]`

Use `help session` for flags and JSON output options.
