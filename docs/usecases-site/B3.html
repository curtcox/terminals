What it does

As a Server (diagnostics engine), I would like to autodetect probable device failures (heartbeat timeout, never-registered, reconnect loop, repeated control errors) and open a pending bug report with the subject's last-known state so that I can surface problems even before a human notices them, and let the first observer confirm the report with one tap.
