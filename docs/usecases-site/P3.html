What it does

As a Developer / power user, I would like to ask the LLM questions, generate code, and have it propose Terminals commands on my behalf, with read-only calls auto-executed and every mutating command gated on my explicit approval so that I can get AI assistance with contextual awareness of my running system without leaving the REPL and without losing control over what runs.
