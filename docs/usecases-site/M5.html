What it does

As a Parent, I would like to have the system watch a room's camera for activity during certain hours so that I can know if someone is where they should (or shouldn't) be.

How to use it

  1. Arm camera monitor on child-room device with window 7:00–8:00 AM.
  2. Camera activity detected at 7:15 AM (inside monitored window — triggers alert).
